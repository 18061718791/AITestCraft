# 前端页面美化 - 共识确认

## 最终需求确认
在不改变项目功能的前提下，建立统一的设计系统，美化所有前端页面，提升用户体验。

## 设计系统规范

### 色彩系统
- **主色**: 蓝色系 (#1890ff - #096dd9)
- **成功色**: 绿色 (#52c41a)
- **警告色**: 橙色 (#faad14)
- **错误色**: 红色 (#ff4d4f)
- **中性色**: 灰度色系 (#f5f5f5 - #262626)

### 字体系统
- **主字体**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- **字号层级**: 12px, 14px, 16px, 20px, 24px, 32px
- **字重**: 400(常规), 500(中等), 600(半粗)

### 间距系统
- **基础间距**: 8px
- **间距层级**: 4px, 8px, 16px, 24px, 32px, 48px

### 圆角系统
- **小圆角**: 4px (按钮、标签)
- **中圆角**: 8px (卡片、模态框)
- **大圆角**: 12px (大卡片、特殊组件)

### 阴影系统
- **一级阴影**: 0 2px 8px rgba(0,0,0,0.08)
- **二级阴影**: 0 4px 12px rgba(0,0,0,0.12)
- **三级阴影**: 0 8px 16px rgba(0,0,0,0.14)

## 实施方案

### 1. CSS变量定义 (src/styles/variables.css)
定义全局CSS变量，统一管理设计token

### 2. Ant Design主题配置 (src/styles/theme.ts)
自定义Ant Design主题，覆盖默认样式

### 3. 全局样式文件 (src/styles/global.css)
统一基础样式和组件样式

### 4. 响应式布局优化
使用Ant Design Grid系统，优化各页面响应式表现

### 5. 组件样式统一
统一按钮、表单、卡片、表格等常用组件样式
